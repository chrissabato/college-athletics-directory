Chrome Extension

Search 1,300+ college athletics websites directly from your browser toolbar.

Download from GitHub
1

Download the repository

Click the button above to download the ZIP file from GitHub. Unzip it somewhere on your computer — you'll need to keep the folder in place.

2

Open Chrome Extensions

In Chrome, navigate to chrome://extensions or open the menu → More Tools → Extensions.

3

Enable Developer Mode

Toggle on Developer mode using the switch in the top-right corner of the Extensions page.

4

Load the extension

Click Load unpacked, then navigate to the unzipped folder and select the extension subfolder inside it.

5

Pin it to your toolbar

Click the puzzle piece icon ⛈ in Chrome's toolbar, find College Athletics Website Directory, and click the pin icon to keep it visible.

Note: This is an unpacked extension and requires Developer mode to remain enabled in Chrome. The extension fetches its data from directory.chrissabato.com and caches it locally for fast searches.
